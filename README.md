# AQUA VOLT
Aqua Volt is a simple and academic website that presents and explains all our team's research throughout the semester. It has many features, like navigation bars, a responsive design, a contact section, and more.
This website explains a micro-hydroelectric project of a Pelton turbine. It presents the research on the grand challenges that we must solve, which are: Improve the Use of Alternative Energies, Recycle Garbage and Waste for Economic and Environmental Purposes, Address & Reduce Pollution in Air, Water, and Soil, Reduce and Adapt to the Effects of Climate Change, and Improve the Use of Water Resources. After that, the problem that we must solve is in the Youssef El-Seddiq center at El-Fayoum governorate. To get the solution, we searched through many prior solutions to combine the points of strength and prevent the points of weakness.
Moreover, we got the solution, which is a Pelton turbine. To know how we built the prototype of the project, the materials used were addressed in a table, and the website explains the methods and how we achieved the design requirements that were set by having a test plan and collecting the results.
Furthermore, this website helps anyone to know more about the Pelton turbine through the analysis, and how they can start searching about it by getting many references in the last part of the body.
